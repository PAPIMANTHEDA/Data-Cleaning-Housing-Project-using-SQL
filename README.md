SQL Data Cleaning Project Report

Overview
This project focused on cleaning and preparing raw data using SQL to ensure accuracy, consistency, and usability for analysis.

Objectives
•	Identify and handle missing or null values 
•	Remove duplicates and inconsistencies 
•	Standardize data formats 
•	Prepare a clean dataset for analysis 

Process
•	Inspected the dataset to understand structure and issues 
•	Removed duplicate records using SQL queries 
•	Handled null values through filtering and replacements 
•	Standardized text fields (e.g., dates, names, categories) 
•	Converted data types for accurate processing 

Tools & Techniques
•	SQL (T-SQL) 
•	Data filtering and transformation 
•	UPDATE, DELETE, CAST, and CONVERT functions 
•	Common Table Expressions (CTEs) 

Outcome
•	Produced a clean, structured dataset ready for analysis 
•	Improved data quality and reliability 
•	Reduced errors and inconsistencies 

Conclusion
This project demonstrates practical SQL data cleaning skills essential for any data analysis workflow, ensuring that raw data can be transformed into a reliable foundation for insights.
